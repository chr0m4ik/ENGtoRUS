# ENGtoRUS
##Преобразователь раскладки текста
Простая утилита для автоматического преобразования текста между английской и русской раскладками.

## ИСПОЛЬЗОВАНИЕ
- При открытии, просто введите строку с неправильной раскладкой, например "Ghbdtn vbh" и программа вам выведет перевод: "Привет мир"

- P.S. не работает с некоторыми символами, например /? в английской раскладке и ., в русской
