# ENGtoRUS
Преобразователь раскладки в тексте из английского в русский(и наоборот).
Использование: при открытии, просто введите строку с неправильной раскладкой, например "Ghbdtn vbh" и программа вам выведет перевод: "Привет мир"
P.S. не работает с некоторыми символами, например /? в английской раскладке и ., в русской
